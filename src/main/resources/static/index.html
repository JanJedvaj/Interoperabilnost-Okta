Validacija i import

                
Ovdje možete importati aplikaciju kroz JSON ili XML. Aplikacija se neće spremiti ako JSON schema ili XML/XSD validacija ne prođe.


                

                    

                        
                        
                    


                    
                    
                    


                    

                        Primjer JSON
                        
{
  "name": "Primjer aplikacijaa",
  "label": "Labela problema",
  "status": "open",
  "signOnMode": "normal",
  "externalId": "test@example.com"
}

                    


                    

                        Primjer XML
                        
<application>
  <name>XML aplikacija</name>
  <label>Labela</label>
  <status>open</status>
  <signOnMode>normal</signOnMode>
  <externalId>test@example.com</externalId>
</application>
